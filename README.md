## Instalação

### Requisitos

* Java 8 *JDK 1.8*
* Maven

### Rodar 
* Rode o comando `mvn spring-boot:run` no prompt de comando para rodar o projeto
* Abra no navegador a url `localhost:8080/pessoa.html`
* Registre uma pessoa
* Registre uma conta bancária
* Faça depositos, saques e tire extratos
